Chart elements

The non-data parts of a chart should recede so the data can speak. NCDPI defaults keep them quiet and consistent.

Anatomy of an NCDPI chart

Every piece of the chart below is set by the brand spec — fonts, sizes, colors, and spacing — so a reader who has seen one NCDPI chart already knows how to read the next one.

Gridlines, axes, and rulers

  • Gridlines are light grey #DEDEDE — present enough to support reading a value, faint enough to disappear behind the data.
  • Axis rulers and dividers use the same light grey. Drop the top and right spines entirely.
  • Reference lines (a target, the state average) are DPI Blue #003A70 so they read as deliberate, not as data.

The zero-baseline rule

Bars and areas encode value by length or area, so they must start at zero. Truncating the axis exaggerates differences and is the single most common way a chart misleads. Lines and dot plots may use a non-zero baseline — but say so if you do.

Labels over legends

When there are seven or fewer items, label them directly rather than making the reader bounce between a legend and the data. Reserve legends for when direct labels would collide.

Declutter, don’t decorate. Every gridline, tick, and label should earn its place. If removing it doesn’t cost the reader anything, remove it.
